Promise

Faz uma promessa enquanto espera o retorno da requisição

    
- ex01

function fazer() {
  let promise = new Promise((resolve, reject)=>{
      setTimeout(()=>{
          resolve('OK');
      }, 3000);
  });
  return promise;
}

fazer()
.then((resposta)=>{
  console.log("Retorno do servidor: "+resposta);
});

Aguarda 3 segundos e dá o retorno.
