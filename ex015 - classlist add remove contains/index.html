classList

Adiciona classe, remove classe, verifica se já tem essa clase

    
- ex01 - Verifica se já tem a class preto na div

document.querySelector("button").classList.contains("preto");



- ex02 - Remove a class "preto" da div

document.querySelector("button").classList.remove("preto");



- ex03 - Adiciona a class "verde" na div

document.querySelector("button").classList.add("verde");
